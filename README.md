# Get Started with Programming Fundamentals to Become a Software Developer

* **Understanding Application Requirements:** Learn the theory and methodology in understanding application requirements from the user side and from the technical specification side of the application. Implement skills into a case study of creating application requirements from the user side and from the technical specification side. Learn work attitude tips when researching, analysing, and evaluating application requirements. 
  
* **Application Modification Planning:** Learn the theory and methodology in creating application requirements, in understanding how applications work, and in understanding flowchart guidelines. Implement skills into case studies of creating application requirements, understanding how applications work, and creating flowcharts. Learn work attitudes when collaboratively planning applications in a team. 
  
* **Understand the Basic Concepts of Programming:** Learn the theory of programming language syntax, variables, data types, computer logic, and JavaScript programming language version ES6. Implement skills by practicing writing pseudocode and writing the first code. Learning work attitude tips in learning programming. 
  
* **Software Application Modification:** Learn HTML5 version of HTML markup language and CSS version 3 programming language. Implementation of skills through a case study of modifying a software application interface. Learn work attitude tips for improving skills in a programming language.

* **Documentation of Software Application Programming and Development:** Learn software archiving theory, code style creation, code commenting, and application technical documentation creation. Implement skills through a case study of archiving a piece of software, adjusting code writing style to fit the standard, adding comments to code, and practice creating application technical documentation. Learn work attitude in communicating documentation to company stakeholders.
